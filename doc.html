Challenges

The challenges of this project was to properly format and display the quiz questions with their answers in a random order. The code would print out the correct answer first and then the following 3 after always so I had to randomize the answers in an array. Which isn't challenging in restrospect but definitely was not friendly for javascript. The array wouldn't properly add the answers at one point and would combine multiple words into one.

Another challenge was tracking the score that the user got from their final sum of answers. Javascript does not seem to play as nicely with variables and constants inside and outside of functions so I was not able to properly keep track after hours and hours of different loops, dispay methods and functions.

Also managing to implement javascript and specific CSS on radio buttons was difficult even with the templates online.
